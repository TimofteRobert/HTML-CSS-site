Mouse

A computer mouse is a hand-held pointing device that detects two-dimensional motion relative to a surface, which is translated to a pointer on a display that allows for smooth control of the graphical user interface on a computer.

Not all operating systems need a mouse, laptops and touch screen capable devices can use them but are not necessary too.
